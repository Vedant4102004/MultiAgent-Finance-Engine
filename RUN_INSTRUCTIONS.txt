🚀 TO RUN THE WEB APP:
═════════════════════

1. Open terminal in project directory

2. Activate virtual environment:
   source venv/bin/activate

3. Run the web app:
   streamlit run web_app.py

4. Browser will open automatically at: http://localhost:8501

❓ TROUBLESHOOTING:

Q: App won't start?
A: Run: source venv/bin/activate
   Then: pip install streamlit

Q: Slow first run?
A: Normal - model loads on first run
   Subsequent runs are faster

Q: No data for company?
A: Check ticker is correct in data_fetcher.py
   Verify company is publicly listed

Q: No news showing?
A: Add FINNHUB_API_KEY to .env
